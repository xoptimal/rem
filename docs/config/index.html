# Config

全局配置

tips: 所有属性方法必须重写实现, 否则影响运行结果

# Attribute

属性 说明 默认值
PROJECT_NAME 项目名称 -
VERSION_CODE 项目代码 -
VERSION_NAME 版本名称 -
PAGE_SIZE 分页数量 10
DEFAULT_LOADING_REFRESH_TYPE 刷新加载方式 1
DEFAULT_LOADING_HANDLE_TYPE 提交加载方式 4
DEFAULT_AUTOMATIC 开启全局自动查询请求 true
DEFAULT_DATA_WAY 全局请求数据格式, 支持类型: json, form-data json
