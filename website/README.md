## Getting Started

 ```bash
    $ mkdir myapp && cd myapp
    $ yarn create umi
    $ yarn install
    $ umi block add https://github.com/xoptimal/rem
```

::: tip
**接下来请移步至[Config](/config/)模块完成配置后就可以正常的使用Rem啦~**
:::
